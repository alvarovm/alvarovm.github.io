== Running ==

For general questions concerning how to run in ALCF computers see:
[[Guides for ALCF]]

Copy [[/home/avazquez/public/gamess/gmsbgq]] and edit to add the name of your project that will be charged.

To  use gmsbgq:
$>./gmsbgq [job name] [nodes] [minutes] [mode] 
 "mode" means number of MPI instances per node, the modes available are: c1, c2, c4, c8, c16 ;
