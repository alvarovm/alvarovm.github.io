====== lgdb ======

To run lgdb, use this wrapper:

   /home/rloy/public/debuggers/lgdb/theta-lgdb

Once you get the lgdb prompt, use the "launch"
command and give it your Cobalt script as a
an argument.

e.g.   launch $a(8) --qsub=myjob.sh hellompi

for an 8-rank run.
